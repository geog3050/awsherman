Description:
Abstract:
This data set represents a digital elevation model (DEM) of the land surface of Johnson County, Iowa, in the UTM projection, Zone 15, NAD83 horizontal datum, with elevation in centimeters NAVD 88 vertical datum. The DEM has a horizontal resolution of 3 meters and was aggregated from one meter resolution elevation data from the state of Iowa's LiDAR program. The aggregation process uses a 3x3 pixel moving average window, which helps to smooth out noise in the one meter data, but also softens sharp edges of landscape features such as ditches and ridges, which may be undesirable for some purposes. The DEM was then Integerizied to shrink the size of the file, so final units are in centimeters. Water features sometimes have a triangular appearance due to lack of lidar returns over water and should be ignored.
Purpose:
This DEM was mainly developed as a basis for creating smooth looking 2' contours. It could also be used in modeling programs requiring a compact representation of the land surface elevation. The suitability for technical, scientific, or finished cartographic purposes is unknown and should not be assumed without thorough testing.

Process_Step:
Process_Description:
Merge one meter bare earth DEM blocks into superblocks covering one or more Iowa counties. Clip using at least a one mile buffer around each county border using ArcMAP data frame and data export command. New file is in ERDAS imagine format, 32 bit float. New file named /countyname/DEM1m.img
Process_Step:
Process_Description:
Create 3 meter resolution grids using spatial analyst/surface/aggregate command, cell factor 3 and aggregation method 'mean'. New file named /countyname/DEM3m.img

Contact_Information:

Process_Step:
Process_Description:
Integerized the grid by using a Times command (x100), then an Add command (0.5), then and Int command. This allows the value to be rounded to the nearest centimeter. Final Grid is a 16-bit Imagine file.
